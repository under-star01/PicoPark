Pico Park 모작 기반의 협동 퍼즐 구조를  
**멀티플레이 프로젝트**로 구현한 Unity 게임입니다.

플레이어는 함께 이동하고, 밀고, 쌓고, 운반하는 협동 요소를 활용해  
스테이지 기믹을 해결하며 목표 지점까지 도달하는 것을 목표로 합니다.

---

## 프로젝트 소개

- **프로젝트명** : Pico Park
- **장르** : 2D 협동 퍼즐 / 액션
- **개발 환경** : Unity, C#
- **플레이 방식** : 협동 멀티플레이

---

## 주요 특징

- 협동 퍼즐 중심의 멀티플레이 구조
- 플레이어 간 밀기, 쌓기, 운반 상호작용 구현
- 스테이지 기믹 해결 중심의 진행 방식
- 협력을 통해 목표 지점에 도달하는 플레이 구성

---

## 실행 방법

- 우측 **Releases**에서 최신 빌드를 다운로드합니다.
- 압축 해제 후 실행 파일을 실행합니다.
